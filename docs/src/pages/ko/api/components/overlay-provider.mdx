# OverlayProvider

`OverlayProvider`는 어디에 오버레이를 렌더링시킬지 결정합니다.

## 인터페이스

```tsx
interface Props {
  children?: ReactNode;
}

export function OverlayProvider({ children }: Props): JSX.Element;
```

## 사용법

일반적으로 오버레이는 다른 요소들 위에 렌더링 되어야 하기 때문에 `<OverlayProvider />`는 React 애플리케이션 루트에 위치합니다.

```tsx {18-21}
import React from 'react';
import { OverlayProvider, overlay } from 'overlay-kit';
import { Modal } from '@src/components';

function App() {
  const notify = () =>
    overlay.open(({ isOpen, close, unmount }) => (
      <Modal isOpen={isOpen} onClose={close} onExit={unmount}>
        {/* ... */}
      </Modal>
    ));

  return <button onClick={notify}>open modal</button>;
}

export function Root() {
  return (
    <OverlayProvider>
      <App />
      {/* 모든 오버레이는 이곳에 렌더링됩니다. */}
    </OverlayProvider>
  );
}
```

> `OverlayProvider`는 React 애플리케이션 전체에서 한 번만 렌더링해야 합니다.
