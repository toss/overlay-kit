# overlay.openAsync

컨펌과 같이 사용자의 입력을 받는 오버레이를 열 때 주로 사용됩니다.

```ts
const result = await overlay.openAsync(controller, options);
```

## 레퍼런스

`overlay.openAsync(controller, options)`

값을 전달받는 오버레이를 열 때 `overlay.openAsync`을 호출하세요.

```tsx
overlay.openAsync(({ isOpen, close, unmount }) => {
  function confirm() {
    close(true);
  }
  function cancel() {
    close(false);
  }

  return <ConfirmDialog isOpen={isOpen} confirm={confirm} close={cancel} onExit={unmount} />;
});
```

### 매개변수

- `controller`: 오버레이 컨트롤러 함수입니다. 이 함수는 오버레이가 열릴 때 호출됩니다.
  - `isOpen`: 오버레이가 열렸는지 여부를 나타냅니다.
  - `close`: 오버레이를 닫는 함수입니다. 인자로 값을 전달해 반환합니다.
  - `unmount`: 오버레이를 제거하는 함수입니다.
- `options`: 오버레이 옵션입니다.
  - `overlayId`: 오버레이를 열 때 고유한 ID를 지정합니다.

### 반환값

`close` 함수로 전달한 값을 반환합니다.

## 사용법

```tsx
const result = await overlay.openAsync(({ isOpen, close, unmount }) => {
  function confirm() {
    close(true);
  }
  function cancel() {
    close(false);
  }

  return <ConfirmDialog isOpen={isOpen} confirm={confirm} close={cancel} onExit={unmount} />;
});

if (result === true) {
  console.log('confirmed');
} else {
  console.log('canceled');
}
```
