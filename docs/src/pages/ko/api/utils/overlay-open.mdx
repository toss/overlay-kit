# overlay.open

경고 또는 노티를 표시하기 위한 오버레이를 열 때 주로 사용됩니다.

```ts
const overlayId = overlay.open(controller, options);
```

## 레퍼런스

`overlay.open(controller, options)`

오버레이를 열 때 `overlay.open`을 호출하세요.

```tsx
overlay.open(({ isOpen, close, unmount }) => {
  return <Alert isOpen={isOpen} close={close} onExit={unmount} />;
});
```

### 매개변수

- `controller`: 오버레이 컨트롤러 함수입니다. 이 함수는 오버레이가 열릴 때 호출됩니다.
  - `isOpen`: 오버레이가 열렸는지 여부를 나타냅니다.
  - `close`: 오버레이를 닫는 함수입니다.
  - `unmount`: 오버레이를 제거하는 함수입니다.
- `options`: 오버레이 옵션입니다.
  - `overlayId`: 오버레이를 열 때 고유한 ID를 지정합니다.

### 반환값

오버레이의 고유한 ID 문자열을 반환합니다.

## 사용법

### 오버레이 열기

```tsx
overlay.open(({ isOpen, close, unmount }) => {
  return <Alert isOpen={isOpen} close={close} onExit={unmount} />;
});
```

### 고유한 ID를 가진 오버레이 열기

```tsx
const overlayId = 'unique-overlay-id';

overlay.open(
  ({ isOpen, close, unmount }) => {
    return <Alert isOpen={isOpen} close={close} onExit={unmount} />;
  },
  { overlayId }
);
```
